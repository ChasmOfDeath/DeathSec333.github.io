# DeathSec333 Security Tools

This repository contains security tools developed by DeathSec333, including:

## Scam Detection Tool
An interactive web-based tool that analyzes text for common scam indicators. The tool can identify:
- Urgency language
- Financial terminology
- Requests for personal information
- Suspicious rewards or offers
- Threatening language
- Cryptocurrency-related scams
- Suspicious URLs and phone numbers

### Features
- Browser-based analysis (no server processing)
- Risk scoring system
- Detailed breakdown of detected indicators
- Customized recommendations based on risk level

## Usage
Visit [https://deathsec333.github.io/scam-detector.html](https://deathsec333.github.io/scam-detector.html) to use the tool directly.
